
# YouTube Music Pop-up Lyrics Extension

A browser extension that displays lyrics in a pop-up window for YouTube Music, inspired by [Spicy Lyrics](https://github.com/Spikerko/spicy-lyrics) and [Better Lyrics](https://github.com/better-lyrics/better-lyrics)
## Lyrics Source

| Provider |
|---|
| **[Better Lyrics](https://github.com/better-lyrics/api)** |
| **[LRCLIB](https://lrclib.net)** |
| **[lrcmux](https://lrcmux.dev/)** |
